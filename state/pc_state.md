## How to read this file

- **HP**: current / max. Major and Severe thresholds shown for reference.
- **Stress**: boxes marked / max. Each marked box is one Stress.
- **Armor**: marked slots / total slots. Marked armor absorbs damage; repaired on long rest.
- **Conditions**: active conditions with source if applicable.
- **Hope**: cross-reference only — authoritative record is `state/hope_fear_ledger.md`.
- **Domain uses**: Domain card abilities that refresh on rest are tracked here.
